## 介绍

致力于开发最全最适用的工具基础标准化，功能包括但不限于快速初始化一个新项目等

## 安装

```
$ npm i @angel6/cli -g

```

## 快速开发

```bash
# 显示帮助信息
$ angel -h

# 显示已支持的模板列表
$ angel list

# 检查cli版本号
$ angel check

# 生成项目结构
# template: 1、vue-webpack   // webpack打包的vue项目
$ angel init <template> <app-name>

```
